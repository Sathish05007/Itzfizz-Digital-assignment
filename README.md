🚗 Scroll Car Animation (GSAP + ScrollTrigger)

An interactive scroll-based animation where a car moves horizontally as the user scrolls vertically.
The animation includes dynamic text reveal, trail effect, and information cards appearing sequentially.

✨ Features
🚗 Car moves left → right on scroll
🟩 Animated road trail follows the car
🔤 "WELCOME" letters reveal based on car position
📦 Info cards appear one-by-one while scrolling
📌 Sticky scroll section (pinned animation)
📱 Fully responsive (mobile + tablet support)
⚡ Smooth animation using GSAP ScrollTrigger

⚙️ How It Works
The page height is increased to create scroll space.
The animation container is pinned using position: sticky.
GSAP ScrollTrigger maps scroll position to animation progress.
The car moves horizontally based on scroll.
The trail width increases dynamically.
Letters appear when the car reaches their position.
Info cards fade in sequentially.
📱 Responsive Design
